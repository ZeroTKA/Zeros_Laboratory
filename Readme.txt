Zeros Laboratory
A personal Unity asset library and testing environment. This is where I build, test, and store modular gameplay systems before dropping them into real projects.

What's in here:
PoolManager / Poolable — Object pooling system
SpawnManager — Coroutine-based enemy spawner with burst, rate, and interval modes
PlayerMovementV2 — First-person character controller (walk, run, crouch, prone, jump)
WeaponData — ScriptableObject for weapon configuration
AmmoHandler — Ammo tracking and reload handling with weapon swap support
Shooting — Raycast and projectile firing with semi, burst, and auto fire modes. 

How to use:
I have included a bunch of documentation within each script. My ultimate goal was to make them as easy as possible to setup within the inspector so knowing code is not a requirement.
